# 数据归一化与标准化


!SLIDE

# 为何要归一化

* 避免大的权值施加决定性影响
* 在传统机器学习里
  * 线性回归, 逻辑回归必须归一化
  * Tree Based可以应对大范围树值
* 与权值初始化同原因

!SLIDE

# 归一化

* 将数值的范围限于 0-1
  * 最大值成 1
  * 最小值成 0

!SLIDE

# 标准化

* 最普遍
* 将数值转换成
  * 平均值 0
  * 标准偏差 1
